# 🍽️ Savor – Premium Food Delivery Platform

Savor is a modern, premium food delivery web application built using HTML, CSS, and JavaScript.  
It provides a complete interactive food ordering experience with cart management, checkout system, customer reviews, and an AI chatbot assistant.

---

## 🚀 Project Overview

Savor simulates a real-world food delivery platform with:

- User login system
- Interactive food menu
- Add-to-cart functionality
- Dynamic cart management
- Checkout & payment options
- Review system
- AI-powered chatbot assistant

This project is designed for learning, portfolio showcasing, and frontend development practice.

---

## 📌 Features

### 🔐 User Login
- Enter username and delivery address
- Smooth animated transition to main app
- Dynamic address display

### 🍕 Interactive Menu
- Filter food by categories (All, Bestsellers, Comfort)
- Beautiful animated food cards
- Add items to cart with instant feedback

### 🛒 Smart Cart System
- Increase/decrease item quantity
- Real-time price calculation
- Slide-in cart drawer
- Dynamic total price update

### 💳 Checkout & Payment
- Credit/Debit Card
- UPI
- Cash on Delivery
- Payment form validation
- Order success animation

### ⭐ Customer Reviews
- Submit new reviews
- 5-star rating system
- Dynamic review rendering

### 🤖 Savor AI Chatbot
- Order assistance
- Food recommendations
- Mood-based suggestions
- Delivery status information
- Address and payment help
- Smart keyword-based intent matching

## 📦 How to Run Locally

1. Clone the repository:

git clone https://github.com/vinodpadimala/savour.git

2. Navigate to project folder:

cd savour

3. Open index.html in your browser.

## 🌐 Deployment (GitHub Pages)

1. Go to your repository on GitHub  
2. Click on Settings  
3. Select Pages  
4. Choose branch: main  
5. Save  

Your project will be live at:

https://vinodpadimala.github.io/savour/
